## Kubernetes tutorials

This repo is the result from me taking notes and following the getting started
with kubernetes on
[pluralsight](https://app.pluralsight.com/library/courses/getting-started-kubernetes) by Nigel Poulton. It may later also feature other courses or learnings from other sources on kubernetes.

### Modules

0 Basics
- Just the basics on what kuberenetes, how it works and some essential concepts
1 Setup
- The basic of setting up and getting started with kubernetes
2 Pods
- Theory and getting started on working with k8s pods
3 Services
- Theory and getting started on working with services
4 Deployments
- Theory and getting started on working with k8s deployments
